Gait API

Complete API endpoint documentation

This page provides detailed technical documentation for the Gait API.

Index

APIDescription
GET http://api.gait.io

Use this endpoint and HTTP method to test if the Gait Server is up and running. This is for testing purposes only, might be deactivated once the app goes live

AccessCode

APIDescription
POST http://api.gait.io/accesscode/validate

Use this endpoint and HTTP method to validate an access code on the Gait Server
GET http://api.gait.io/accesscode

Use this endpoint and HTTP method to retrieve the access code from the Gait Server. This is for testing purposes only, might be deactivated once the app goes live

Acceleration

APIDescription
POST http://api.gait.io/users/{userid}/accelerations

Use this endpoint and HTTP method to create an acceleration for a user on the Gait Server

User

APIDescription
POST http://api.gait.io/users

Use this endpoint and HTTP method to create a user on the Gait Server
PUT http://api.gait.io/users/{userid}

Use this endpoint and HTTP method to update a user on the Gait Server
GET http://api.gait.io/users/{userid}

Use this endpoint and HTTP method to get a user from the Gait Server
GET http://api.gait.io/users/email/{email}

Use this endpoint and HTTP method to get a user from the Gait Server by email
POST http://api.gait.io/users/login

Use this endpoint and HTTP method to login a user on the Gait Server
POST http://api.gait.io/users/{userid}/report

Use this endpoint and HTTP method to report an offensive user on the Gait Server

ExtraInfo

APIDescription
GET http://api.gait.io/users/{userid}/extrainfo

Use this endpoint and HTTP method to get the extra info for a user from the Gait Server
PUT http://api.gait.io/users/{userid}/extrainfo

Use this endpoint and HTTP method to update the extra info for a user on the Gait Server

Match

APIDescription
GET http://api.gait.io/users/{userid}/match

Use this endpoint and HTTP method to get the match for a user from the Gait Server
POST http://api.gait.io/matches

Use this endpoint and HTTP method to create a match between 2 users on the Gait Server. This is for testing purposes only, will be deactivated once the app goes live

Chats

APIDescription
GET http://api.gait.io/users/{userid}/chats

Use this endpoint and HTTP method to get the active chats for a user from the Gait Server
PUT http://api.gait.io/users/{userid}/chats/{receiverid}?action

Use this endpoint and HTTP method to update (block/unblock/delete) a private chat for a user on the Gait Server

Message

APIDescription
POST http://api.gait.io/users/{userid}/messages/{receiverid}

Use this endpoint and HTTP method to create a message for a user in a private chat on the Gait Server
GET http://api.gait.io/users/{userid}/messages/{receiverid}?startdate

Use this endpoint and HTTP method to get the messages for a user in a private chat from the Gait Server
DELETE http://api.gait.io/users/{userid}/messages

Use this endpoint and HTTP method to delete messages for a user in a private chat from the Gait Server

Activity

APIDescription
POST http://api.gait.io/users/{userid}/activities

Use this endpoint and HTTP method to create an activity for a user on the Gait Server
GET http://api.gait.io/users/{userid}/activities

Use this endpoint and HTTP method to get the last activity for a user from the Gait Server

Interest

APIDescription
POST http://api.gait.io/interests

Use this endpoint and HTTP method to register an interest for a user to use the app on the Gait Server
